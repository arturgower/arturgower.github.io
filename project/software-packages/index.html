Packages

We write high performing packages, mostly in Julia, related to acoustics and elastic waves, and more generally inverse problems from boundary measurements, such as stress and thermal predictions.

MultipleScattering.jl

Under close inspection, many natural and synthetic materials are composed of small randomly distributed particles. To measure these particles we need to predict the multiple scattering of waves between the particles.

View Project on GitHub →

EffectiveWaves.jl

Calculates ensemble averaged waves in complex materials. A type of homogenisation method very useful for sensing.

View Project on GitHub →

ElasticWaves.jl

A package to calculate propagation and scattering of elastic waves in bearings. A great setting for inverse problems!

View Project on GitHub →

Entropy GO

To calculate the current influence on a Go board we use entropy maximisation which is often used for intelligent decision-making.

View Project on GitHub →
